## ğŸ’¡ Solution

**Community Problem Solver** is a web-based platform that allows users to:

* Report real-world community problems with a picture
* Categorize issues based on type
* Discuss possible solutions with others
* Issues are solved by people or government
* Upvote important problems to increase visibility
* Downvote if issue is not solved properly

This creates a collaborative, transparent, and community-driven approach to solving real-life problems.
